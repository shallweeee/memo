jira 티켓을 복사하는 예제

## Setup
```
$ conda create -n jira python=3.8.5
$ conda activate jira
$ pip install -r requirements.txt
$ vi config.py  # 계정 정보 설정
```

## Run
```
$ python clone.py
```
